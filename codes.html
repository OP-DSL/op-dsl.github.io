OP DSL - Codes

This page lists a number of applications that are build using the OPS and OP2 DSLs.

OP2

  • Rolls-Royce Hydra - turbomachinery design code, written in Fortran
  • VOLNA - non-linear shallow water equations solver, written in C.
  • BookLeaf - 2D Arbitrary Lagrangian-Eulerian FEM code, written in Fortran. Part of UK-MAC.
  • BASEMENT - river morphology simulator, being developed at ETH ZÃ¼rich, written in C and Python.
  • MG-CFD - 3D unstructured multigrid, finite-volume computational fluid dynamics (CFD) mini-app for inviscid-flow.

OPS

  • OpenSBLI - A framework for the automated derivation of finite difference solvers from high-level problem descriptions.
  • UK-MAC Mini-Apps : CloverLeaf 2D (2D Structured mesh hydrodynamics), CloverLeaf 3D (3D Structured mesh hydrodynamics) and TeaLeaf 2D (2D Structured mesh sparse linear solvers).
  • HiLeMMS: High-Level Mesoscale Modelling System, developed at STFC.
