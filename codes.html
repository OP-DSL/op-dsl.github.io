OP DSL - Codes

This page lists a number of codes that build on the OPS and OP2 libraries

OP2

  • Airfoil - non-linear 2D inviscid code with Finite Volume Discretisation. Ships with OP2, C and Fortran variants.
  • Aero - 2D Finite Element code. Ships with OP2, C version.
  • Rolls-Royce Hydra - turbomachinery design code, written in Fortran
  • VOLNA - non-linear shallow water equations solver, written in C. Released soon
  • BookLeaf - 2D Arbitrary Lagrangian-Eulerian FEM code, written in Fortran. Part of UK-MAC: Link
  • BASEMENT - river morphology simulator, being developed at ETH ZÃ¼rich, written in C and Python. OP2 version to be released soon. Link

OPS

  • Poisson, MBlock, multiDim, ADI - simple applicaitons demonstrating the features. Ship with OPS, C and Fortran variants
  • CloverLeaf 2D - 2D Structured mesh hydrodynamics code. Ships with OPS, C variant. Part of UK-MAC: Link
  • CloverLeaf 3D - 3D Structured mesh hydrodynamics code. Ships with OPS, C variant. Part of UK-MAC: Link
  • TeaLeaf 2D - 2D Structured mesh sparse linear solvers. Ships with OPS, C variant. Part of UK-MAC: Link
  • Lattice-Boltzmann code, developed at STFC by Jianping Meng
